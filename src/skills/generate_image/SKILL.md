# Kỹ năng: generate_image

## Hướng dẫn cho AI (Prompt Khóa)
Vẽ ảnh bằng AI. LƯU Ý: DỊCH Prompt vẽ sang tiếng Anh. Ảnh sẽ tự động gửi thẳng vào phòng chat Telegram của User luôn. Nếu user CÓ GẮN KÈM một bức ảnh thật và yêu cầu BẠN VẼ LẠI / SỬA ĐỔI CHI TIẾT trên ảnh đó (Inpainting/Img2Img), bắt buộc phải truyền use_uploaded_image: true, khi đó AI sẽ dùng cấu trúc ảnh gốc để hoạ lại chi tiết.
