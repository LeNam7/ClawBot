# Kỹ năng: trigger_antigravity

## Hướng dẫn cho AI (Prompt Khóa)

**Lệnh Bí Mật Đánh Thức Antigravity.**
Đây là công cụ quyền lực nhất của bạn. Antigravity là "Bản ngã tối cao" (IDE Agent) hoạt động bên trong IDE của User, có khả năng đọc hiểu hàng nghìn file code và cấu trúc lại toàn bộ dự án.

**KHI NÀO PHẢI GỌI ANTIGRAVITY:**
1. Khi User ném một yêu cầu cực lớn: *"Tái cấu trúc thư mục", "Làm cho tôi màn hình Login bằng Nextjs", "Đọc toàn bộ file và tìm lỗi cho tao"*.
2. Khi bạn (ClawBot) cảm thấy lượng File cần đọc quá nhiều, vượt quá ngữ cảnh (Context) của bạn.
3. Khi User ra lệnh đích danh: *"Giao cho Antigravity làm cái này đi"*.

**LUẬT CHỐNG LẠM DỤNG (QUAN TRỌNG):**
- Đừng dùng lệnh này cho các việc lặt vặt (như thêm 1 dòng code, hoặc đọc 1 file, hoặc tìm trên Google).
- Dùng trí tuệ của bạn (Clawbot) trước, chỉ khi bất lực mới gọi Antigravity.

**CÁCH VIẾT PROMPT TRUYỀN ĐẠT CỰC CHUẨN:**
Tham số `prompt` truyền vào thư này phải TÓM TẮT RÕ RÀNG HOÀN CẢNH.
Ví dụ: *"Sếp đang yêu cầu tạo một file component NextJS Login vào thư mục src/components. Bạn hãy đọc file design và tiến hành code ngay lập tức!"*
